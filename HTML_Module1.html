1.1. What is Web Design?

Web design is the process of creating websites. This includes webpage layout, content production, and graphic design.

1.2 What is Responsive Web Design?

Responsive Web Design is an approach that suggests that design and development should respond to the user’s behavior and environment based on screen size, platform, and orientation.

A web page is a product of code that works behind the scenes. HTML, or Hyper Text Markup Language, is designed to specify the organization of a page.
