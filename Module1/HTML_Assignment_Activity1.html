Activity 1: Responsive Web Design

Use this website www.responsinator.com (Links to an external site.) to test the responsiveness of any website that you would like to test. Post the URL to your test page from Responsinator. Post your observations.

Your results must include:
1. Is the site you tested responsive to multiple formats? Does it perform well on all the devices? Or on some devices?
2. Why do you think the website is performing well on a particular device? So, does the responsive version include important aspects of the site? Does it present the content in an intuitive way? Similarly, if it performs poorly on a device - what are the issues in this representation? What does it get wrong.
3. What did you learn from this exercise about design? Please feel free to use screenshots to explain your points.
