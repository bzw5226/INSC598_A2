Assignment 2: Static Website Creation

Overview

For this assignment, you will be creating a static html site with multiple pages. The purpose of this assignment is to provide you with hands on experience with html, css, site navigation and creation of a static website.

Identifying a topic for the Website

I am providing you with some suggestions for creating a website, these are just suggestions, if there is any other information environment for which you would like to create a site, please email me with your topic of choice.

  1. An organization that you work for
  2. A website for a personal e-portfolio
  3. A website for this course
  4. A website for an interest group
  5. A resource website like a pathfinder or a collection of tutorials on a topic and its related sub-topics, etc.

Details of the Website

For successful completion of this assignment you will be implementing the following:

Hosting

Your website will be hosted on the utk server, you will be using the server space provided to one your group members. Identify that site as the first thing in the process.

Website Features

  1. Each website will consist of at least nine html pages. These pages are to be accessible from homepage and must be interlinked.
  2. Website must have a logo or a banner at the top of homepage and on all the pages.
  3. All the pages on the website must have identical navigation on all the pages allowing you to navigate between pages.
  4. The webpages must be some other font than the default and the color of the non-link text in the document should be some other color than black.
  5. The pages on the site should have a background color on other than white.
  6. There should be external hyperlinks on your website that direct you to a website that is external.
  7. The links should change color when clicked; mouse hover and the color should be something other than the default color.
  8. Your pages should validate using the W3C HTML Validator. Warnings are OK (yellow) - your documents cannot have any errors (red).
  9. You will submit the assignment in the form of a URL.

Grading

This assignment is 30% of your grade, 25% of that will be based on the website submitted and 5% will be based on the peer evaluation that your team will submit for each of the group member. The details for the peer evaluation will be provided to you.
