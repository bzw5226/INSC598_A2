Activity 4.2: Practicing CSS

Hi All,

If you would like to practice how to make CSS work; here is an optional exercise for you.

This will help you in understanding how to apply a CSS file to your HTML files.

For this activity you will need atleast two HTML files (from a previous assignment or workshop will work fine or from your Assignment 2 content) and you will create one css file.

Important to have ALL the files - HTML, images and the CSS file in one folder on your desktop.

Start simple, by applying one rule in your css file â€“ for example

1. you could write a rule about the page background color in a text file

body {
background-color: yellow;
}

2. then save this text file as a styles.css

3. Then add the following line in each HTML page
"link rel="stylesheet" href="styles.css"

4. Now you have applied the style sheet to all HTML files where you added that line above

5. Now you can edit the styles.css file one rule at a time for as many rules as you need

So, for your Assignment 2 website - my recommendation is to first make your HTML pages with the content that you want in these pages and then create an external style sheet and apply it to these pages.

Hope this helps.
